### 연구 내용(목적)
1. App 사용량과 미세먼지 포함 날씨와의 주요 상관관계 확인(2019년 App log data)

  - 이외의 날씨 정보와 날짜 정보(월, 휴일 여부 등)를 통해서 통계적 분석
